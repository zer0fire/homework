快速幂算法
log 以 2 为底的对数可以算 x 的 n 次幂，思路是根据 x^n 可以拆分成 x^2 的 二进制形式，比如 x^1024 -> x^512 \* x^512 -> ... x^2 \* x^2
x^1025 -> x^1024 \* x^1

```js
const pow = (v, n) => {
  let r = v,
    m = n - 1,
    c = v;
  while (m > 0) {
    if (m % 2) {
      r = r * c;
    }
    m = m >> 1;
    c = c * c;
  }
  return r;
};
```
